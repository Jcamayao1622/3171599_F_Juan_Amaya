# 🔄 Week 03 - useEffect

Dashboard básico que consume datos desde una API usando useEffect.

- Obtener datos desde API
- Mostrar datos en lista
- Actualización automática (polling)
- Manejo de loading y error

## 🧩 Conceptos Aplicados

- useEffect
- Fetch API
- Estados (loading, error, data)
- Cleanup con clearInterval
