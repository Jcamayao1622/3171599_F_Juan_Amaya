# ⚛️ Week 02 - React CRUD

## 🧠 Descripción

Aplicación CRUD en React con TypeScript para gestionar maquinaria agrícola.

## 🚀 Funcionalidades

- Agregar maquinaria
- Editar maquinaria
- Eliminar maquinaria
- Listar maquinaria

## 🧩 Conceptos Aplicados

- useState
- Formularios controlados
- Renderizado con `.map()`
- Eventos (onClick, onChange, onSubmit)
- Inmutabilidad

## ▶️ Ejecución

```bash
pnpm dev
